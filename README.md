## 💸 Pricing Strategy (Customer Psychology)

| Plan | Original | Discount | Price | Per Day | Psychology |
|------|----------|----------|-------|---------|------------|
| Starter (15 days) | ₹1,299 | 38% OFF | ₹799 | ₹53/day | Low-risk trial |
| **Monthly (30 days)** ⭐ | **₹2,599** | **43% OFF** | **₹1,499** | **₹50/day** | Best value anchor |
| Combo (90 days) | ₹7,797 | 49% OFF | ₹3,999 | ₹44/day | Maximum savings |

### Psychology Tactics Used:
- ✅ **Anchoring**: Original ₹2,599 makes ₹1,499 feel like a steal
- ✅ **Decoy Effect**: Starter ₹799 makes Monthly ₹1,499 look best value
- ✅ **Per-Day Pricing**: ₹50/day sounds cheaper than ₹1,499/month
- ✅ **Scarcity**: "सिर्फ 23 पैक बाकी!" countdown
- ✅ **Social Proof**: 52,847+ customers, live visitor counter
- ✅ **Urgency**: Countdown timer, stock counter
- ✅ **Authority**: Made in Germany, Lab Certified, Doctor endorsement
- ✅ **Risk Reversal**: COD, 7-day return, free delivery
- ✅ **FOMO**: Sales alerts showing others buying

## 📱 Google Sheets Setup

1. Create new Google Sheet named "SugarDefend Pro Leads"
2. Extensions > Apps Script
3. Paste `google-apps-script.js` content
4. Run `setupSheet()` function once
5. Deploy as Web App (Anyone access)
6. Copy URL → paste in `index.html` line: `const SHEET_URL = 'YOUR_URL'`

### Sheet Columns:
| Timestamp | Name | Phone | Plan | Source | Status | Notes | Follow-up |

### Status Options:
- New → Called 1 → Called 2 → WhatsApp Sent → Order Confirmed → Delivered

## 🔧 Customization

### Change Phone Number
Search for `9876543210` and replace with your number.

### Change Product Images
Replace files in `uploads/` folder.

### Change Pricing
Edit values in pricing section (search for `₹1,499`).

### Add YouTube Video
Replace `dQw4w9WgXcQ` with your video ID in the `playVideo` function.

### Change Colors
Edit CSS variables at top of `<style>`:
```css
--orange: #FF6B00;   /* Main CTA color */
--green: #1B5E20;    /* Trust color */
--red: #C62828;      /* Urgency color */
```
